# shupian-APP
薯片APP——一款属于大学高校自己的约拍APP
功能:
1.用户账号注册模块
2.用户登录模块
3.Show界面
(1)以瀑布流的形式向用户展现摄影师们上传的精美图片
(2)推送最新的咨询给用户，可以分享摄影技巧、摄影经验或者摄影成果等文章
4.Photographer界面
(1)展示所有本平台注册的摄影师并且用户可以搜索认识的摄影师
(2)用户可以查看摄影师们的主页，了解、关注、联系摄影师
(3)用户可以查看摄影师们的图库，查看他们分享的照片和往期影集
5.Message界面
(1)用户可以查看自己关注的摄影师列表
(2)用户可以和自己关注的摄影师即时聊天
6.Setting界面
(1)更改用户头像
(2)显示用户个人信息
(3)设置用户所在城市和所在大学信息
(4)成为薯片平台摄影师
(5)查看用户自己的相册，上传图片到自己的相册，创建新的相册
